# Piano Virtual

Um simples piano virtual desenvolvido com JavaScript, HTML e CSS.

## Funcionalidades

- Toque notas musicais ao clicar nas teclas do piano ou do seu teclado.
- Interface amigável e responsiva.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
